## ©basic a. Soma 2


Escreva uma função soma que recebe dois parâmetros e devolve a soma dos dois parâmetros.

## Main

```hs
main = do
    x <- readLn::IO Int
    y <- readLn::IO Int
    print $ soma x y
```
