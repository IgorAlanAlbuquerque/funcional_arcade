## Compactador de vetores

Dado um vetor, compacte os elementos consecutivos que se repetem numa lista de listas.


```
>>>>>>>>
[]
========
[]
<<<<<<<<

>>>>>>>>
[1]
========
[[1]]
<<<<<<<<

>>>>>>>> um duas vezes
[1,1]
========
[[1,2]]
<<<<<<<<

>>>>>>>>
[1,1,1]
========
[[1,3]]
<<<<<<<<

>>>>>>>>
[1,1,1,5]
========
[[1,3],[5]]
<<<<<<<<

>>>>>>>>
[1,1,1,5,5]
========
[[1,3],[5,2]]
<<<<<<<<

>>>>>>>>
[1,1,1,5,5,0]
========
[[1,3],[5,2],[0]]
<<<<<<<<

>>>>>>>>
[1,1,1,5,5,0,5,5]
========
[[1,3],[5,2],[0],[5,2]]
<<<<<<<<

```
